初用 LeanCloud - 新建一个对象

发表于 2017-07-21 | 分类于 LeanCloud

创建完成后，打开 控制台 > 存储，点开 Todo ，就可以看到刚才添加的数据。

内置属性 类型 描述
objectId String 该对象唯一的 Id 标识
ACL ACL 该对象的权限控制，实际上是一个 JSON 对象，控制台做了展现优化。
createdAt Date 该对象被创建的 UTC 时间
updatedAt Date 该对象最后一次被修改的时间
